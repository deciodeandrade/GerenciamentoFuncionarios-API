# Começando

Após clonar o projeto na sua máquina... Através do terminal navegue até o diretório do projeto e...

Execute: bundle

Execute: rails db:create

Execute: rails db:migrate

Para subir o servidor da API...
Execute: rails s -p 3001

Para acessar a API: 
http://localhost:3001/
